Rapid Web App Dev: Event Booking Platform

Overview:

This project involved designing and building a functional event booking platform. Initially developed as a solo concept, the project was selected (top grade in the class) to be used for the group build phase. The final product included complete user authentication, event browsing, bookings, and admin event management.

Tech Stack: HTML, CSS, JavaScript, PHP, MySQL

My Role:

  • Designed and prototyped the full user flow and UI/UX during the solo phase
  • Led group development after my design was selected as the foundation
  • Implemented booking logic, admin dashboard, and front-end styling
  • Collaborated using GitHub for source control and issue tracking

Key Features:

  • User registration and login system
  • Event search, filtering, and booking functionality
  • Admin portal for managing events and bookings
  • Responsive front-end with a clean, modern interface
