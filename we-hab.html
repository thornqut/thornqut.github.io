We-Hab: Wellness & Recovery App

Overview:

We-Hab is a Java-based desktop application designed to support rehabilitation patients in tracking their daily recovery routines. The app allows users to log medication, mood, and exercise data, which can then be compiled into reports for medical professionals.

Tech Stack: Java, JavaFX, FXML, SQLite, SceneBuilder, GitHub, Jira

My Role:

  • Led the FXML UI design using SceneBuilder to create a clean, user-friendly interface
  • Managed GitHub repository structure, branches, and merges
  • Contributed to team meetings, agile planning, and bug resolution
  • Wrote and presented the GitHub & user story overview video

Key Features:

  • Log and track daily mood, medications, and workouts
  • Exportable reports for health professionals
  • Clean, accessible UI design
  • Flexible reminders and activity scheduling
