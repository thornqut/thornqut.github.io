Capstone: CamVision AI – Intelligent CCTV Monitoring System

Overview:

As part of my final-year capstone project, my team collaborated with an industry partner to design a system for monitoring the performance and quality of commercial CCTV cameras. Our goal was to architect an AI-powered solution that detects camera issues (e.g., blur, tampering, downtime) in real time and automatically triggers alerts or maintenance actions.

Status: Phase 1 (Planning & System Design) – Completed

My Role:

  • Designed the initial AWS-based architecture for data ingestion, model hosting, and alerting
  • Set up the team's GitHub repository and CI/CD planning documentation
  • Created and structured the Jira board for task management and sprint planning
  • Led documentation of the AI model pipeline (YOLOv8, PyTorch), proposed integrations with AWS S3, EC2, and SQS

Tools & Technologies:

  • GitHub, Jira, Confluence, Miro
  • AWS EC2, S3, ECR, SQS (planned)
  • YOLOv8, PyTorch, Docker

Key Deliverables (so far):

  • System architecture diagram and proposed cloud deployment plan
  • Team charter, sprint schedule, and Confluence documentation
  • ML model research and edge device viability analysis

Note: Due to client confidentiality, code, diagrams, and detailed documentation are not publicly available.
