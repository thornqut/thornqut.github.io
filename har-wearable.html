Cloud API: Dockerized Web Application on AWS

Overview:

This project involved deploying a containerized REST API to AWS EC2 using Docker and Amazon ECR. The application handled basic user authentication using JWT and executed CPU-intensive operations. It was designed to be stateless, scalable, and production-ready using cloud-based infrastructure and best practices.

Tech Stack: Python (Flask), Docker, AWS EC2, ECR, S3, JWT, GitHub, Linux CLI

My Role:

  • Designed and implemented the full deployment pipeline using Docker & AWS
  • Configured AWS EC2 instance with firewall rules, SSH access, and Linux environment
  • Containerized the REST API and pushed image to Amazon ECR
  • Wrote and demoed the CPU-intensive endpoint with logging & JWT protection

Key Features:

  • RESTful API endpoints secured with JWT tokens
  • Stateless architecture using cloud storage and session-less logic
  • Custom Docker image with CPU-bound background task
  • Deployed via EC2 and tested for scalability and fault tolerance
